MODS


    

      ⚠
      

        Only load stylesheets and scripts you trust.
        Extensions run with full access to the Gobbonet interface.
        External URLs are fetched at load time — keep them on your local network
        or from sources you control.
      

    


    

      
      

        
        Toggle all extensions on/off without clearing them.
      

    


    
    

      
🎨 STYLESHEETS

      

        Load one or more stylesheets. Each entry can be a URL, inline CSS, or both
        (inline applies after its URL). They're injected in order, top to bottom.
      

      

      

        
        
      

    


    
    

      
⚡ SCRIPTS

      

        Load one or more scripts. Each entry can be a URL or inline JS. They run in
        order at boot with full DOM + state access.
      

      

      
    


    
    

      
● CUSTOM MACROS

      

        

          Define {{trigger}} shortcuts — when found in any message or character card field, the trigger expands to your text before being sent to the model. In the chat itself the compact {{trigger}} tag stays visible, so your messages don't get cluttered with the full expansion.


          Built-in (read-only): {{char}} · {{user}} · {{current_DAT}} (these still resolve on screen)
